Installing Bower

$ npm install --global bower
$ bower --version && bower help

Installing Bower Packages

$ bower install jquery
$ bower install <source>#<target> [--save]
# <source> = registry name, github, git URL
# <target> = semver, branch, sha1
$ bower list --paths

Finding Bower Packages

$ bower search bootstrap
$ bower home bootstrap

Alternatively:

  • http://bower.io/search/

Local Package Development

$ bower link
$ bower link <myownpackage>
$ bower version <whattobump>
$ git push --tags

Inspired by:

  • https://oncletom.io/2013/live-development-bower-component/

Summary

$ bower init     # generating bower.json
$ bower install  # installing packages
$ bower list     # listing local packages
$ bower search   # finding packages
$ bower link     # local development
